## 0.0.1

Initial custom fork release:

- Uses a unique plugin ID (`kanban-bases-view-cl`) and Bases view type (`kanban-view-cl`) so it can coexist with the upstream Kanban Bases View plugin.
- Adds custom card image gallery support for list properties containing multiple images.
- Adds per-group column collapse controls.
- Adds configurable column width options.
- Keeps the upstream kanban Bases view features: drag-and-drop cards, quick add buttons, swimlanes, column ordering, column colors, custom card titles, cover images, and property display options.
